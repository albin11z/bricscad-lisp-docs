V.1.1

Additions & Changes with V.1.1 (BricsCAD V18.2)



  • BricsCAD V18.2 LISP engine supports VLX NameSpace feature for DES files created by new DESCoder v2.7;
    For more details, see topic VLX NameSpace Support.


  • BricsCAD V18.2 LISP engine provides the new integrated LISP Editor & Debugger IDE, with many enhanced functionality to edit, handle and debug LISP applications and projects.


  • added documentation for BricsCAD specific details related to DCL function support


  • extended the BricsCAD BIM LISP interface for new Building Element Types

    now all properties of BIM objects (as shown in PropertiesPanel) are accessible via LISP BIM functions;

    Since V18.2, the LISP BIM interface supports "categorised" properties; if a category is not specified, the default bcBimCategoryStandard is used, which combines the bcBimCategoryBricsys and bcBimCategoryIFC2x3 categories, compatible with V17.

    Additionally, the bcBimCategoryIFCCustom and bcBimCategoryUser categories allow to attach custom data (to IFC namespace resp. plain "user" namespace) to any BIM classified object.


  • extended the BricsCAD BIM LISP interface to support new "Rooms" functionality

    Since V18.2, the LISP BIM interface supports the "Rooms" functionality to handle BIM "Rooms".


  • added more DOSLib emulation functions :

dos_command

dos_fileex
