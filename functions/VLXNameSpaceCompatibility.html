VLX NameSpace Compatibility


Since DEScoder.exe v2.7 :

The "VLX NameSpace" feature to isolate Lisp code running in an own "NameSpace" (prevents clashes of same-named functions, between different Lisp files).is available since DEScoder v2.7, and supported since BricsCAD V18.2.

But the internal DES data format and options storage has been made backward + forward compatible, to provide best level of compatibility possible.
