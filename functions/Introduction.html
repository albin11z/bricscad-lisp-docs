Recent Web Browser versions (Firefox, Chrome, ...) are trying to increase security of user computers;
one action was to disable "cross-origin" web content loading - LDSP is affected, though our help system here uses all html files from *same* location.

How to re-enable this navigation in your browser :

Firefox

about:config => privacy.file_unique_origin => change to "false"
in more recent Firefox version, this changed to
security.fileuri.strict_origin_policy => change to "false"

Chrome

chrome://flags =>then search for "Cross Origin Embedder Policy" => change to "Disabled"

Others

(Edge, IE 11, Opera, Safari), please check the web how to disable this new "feature".
