vle_g_vecTol

vle_g_vecTol


vle_g_vecTol

double 1.0e-10

Preset tolerance for (vle-vector-xxx) functions.

Lisp code can retrieve and adjust to a different value if needed :

(setq orgTol (vle-vector-getTolerance))
(vle-vector-setTolerance  0.00000001)  ;; preset as 1.0e-08

Remarks : 

Please do not access ''vle_g_vecTol" directly, always use (vle-vector-getTolerance) and (vle-vector-setTolerance) !
